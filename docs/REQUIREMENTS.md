# 三十六計 〜天下統一への道〜 要件定義書

## 1. プロジェクト概要

### 1.1 ゲームタイトル
**三十六計 〜天下統一への道〜**
（英語: Thirty-Six Stratagems - Road to Unification）

### 1.2 コンセプト
中国古典兵法「三十六計」を題材とした戦略シミュレーションゲーム。
プレイヤーは一国の君主として、36の計略を学び・駆使しながら天下統一を目指す。

### 1.3 ターゲット
- 一般向けカジュアルゲーマー
- 戦略ゲーム好き
- 中国史・兵法に興味がある人

### 1.4 プラットフォーム
- PC (Windows)
- 将来的にMac/Linux対応も視野

### 1.5 開発環境
- Unity 2022.3 LTS 以上
- C#
- Git/GitHub

## 2. ゲームシステム要件

### 2.1 ゲームモード

#### 2.1.1 キャンペーンモード（メイン）
- 全6章構成のストーリーモード
- 各章で1カテゴリ（6計略）を習得
- 章ごとにボス戦（習得した計略の応用が必要）
- クリア後に次章解放

#### 2.1.2 フリーモード
- 全計略解放状態でプレイ
- マップサイズ: 小(8領地) / 中(12領地) / 大(20領地)
- 勢力数: 2〜6
- CPU難易度: 易 / 普通 / 難 / 極難

#### 2.1.3 計略図鑑
- 36計すべての解説
- 原典の漢文 + 現代語訳
- 歴史上の使用例
- ゲーム内効果説明
- 解放状況の表示

### 2.2 基本ゲームフロー

```
ゲーム開始
    ↓
ターン開始
    ↓
┌─────────────────────────────────┐
│ フェーズ1: 内政フェーズ          │
│   - 資金収入                    │
│   - 兵士徴兵                    │
│   - 施設建設                    │
│   - 武将配置                    │
└─────────────────────────────────┘
    ↓
┌─────────────────────────────────┐
│ フェーズ2: 外交フェーズ          │
│   - 同盟申請/破棄               │
│   - 停戦交渉                    │
│   - 計略使用（外交系）           │
└─────────────────────────────────┘
    ↓
┌─────────────────────────────────┐
│ フェーズ3: 軍事フェーズ          │
│   - 軍の移動                    │
│   - 戦闘                       │
│   - 計略使用（軍事系）           │
└─────────────────────────────────┘
    ↓
ターン終了 → 敵ターン → 次ターンへ
```

### 2.3 マップシステム

#### 2.3.1 領地
- 各領地は1つの勢力が支配
- 領地パラメータ:
  - 人口: 徴兵上限に影響
  - 経済力: 収入に影響
  - 防御力: 城の守りやすさ
  - 隣接領地: 移動可能な領地

#### 2.3.2 マップ構成
```
領地A ─── 領地B ─── 領地C
  │         │         │
領地D ─── 領地E ─── 領地F
  │         │         │
領地G ─── 領地H ─── 領地I
```
（実際はより複雑なネットワーク構造）

### 2.4 リソースシステム

| リソース | 説明 | 用途 |
|---------|------|------|
| 金 | 通貨 | 徴兵、建設、計略発動 |
| 兵糧 | 食料 | 軍の維持、遠征 |
| 兵力 | 兵士数 | 戦闘力の基盤 |
| 計略ポイント(CP) | 計略発動用 | 計略使用時に消費 |

### 2.5 戦闘システム

#### 2.5.1 戦闘の流れ
1. 攻撃側が進軍を宣言
2. 両軍の兵力・武将能力を比較
3. 計略使用の選択
4. 戦闘解決（自動計算 or 簡易演出）
5. 結果表示（勝敗、損害）

#### 2.5.2 戦闘計算式（基本）
```
戦闘力 = 兵力 × (1 + 武将能力/100) × 地形補正 × 計略補正
勝敗判定 = 攻撃側戦闘力 vs 防御側戦闘力
損害率 = 敗者は30-50%、勝者は10-20%の兵力損失
```

### 3.2 計略ポイント（CP）システム

- 最大CP: 軍師の知力に依存（基本10）
- CP回復: 毎ターン開始時に知力/20ポイント回復
- 消費CP: 計略により1〜5ポイント

### 3.3 計略発動条件

各計略には発動条件がある:
- 状況条件（優勢時のみ、劣勢時のみ等）
- 対象条件（敵勢力、同盟勢力等）
- リソース条件（CP、金等）

## 4. キャラクターシステム

### 4.1 キャラクター種別

| 種別 | 役割 | 主要能力 |
|------|------|----------|
| 君主 | 勢力の長 | カリスマ、決断力 |
| 軍師 | 計略担当 | 知力、CP回復 |
| 武将 | 戦闘指揮 | 武力、統率 |
| 間者 | 諜報担当 | 諜報、計略補助 |

### 4.2 能力値

| 能力 | 説明 | 影響 |
|------|------|------|
| 武力 | 戦闘の強さ | 戦闘力ボーナス |
| 知力 | 頭の良さ | CP回復、計略成功率 |
| 統率 | 兵を率いる力 | 指揮可能兵力上限 |
| 政治 | 内政能力 | 領地収入ボーナス |
| 魅力 | 人を惹きつける力 | 登用、外交成功率 |

### 4.3 武将の得意計略

各武将は特定の計略カテゴリが得意:
- 得意カテゴリの計略: 成功率+20%、消費CP-1
- 不得意カテゴリの計略: 成功率-10%

## 5. 勝敗条件

### 5.1 勝利条件
- **制覇勝利**: 全領地を支配
- **降伏勝利**: 全敵勢力が降伏
- **盟主勝利**: 同盟勢力の合計領地が全体の70%以上

### 5.2 敗北条件
- 本拠地の陥落
- 君主の死亡（捕縛後処刑）
